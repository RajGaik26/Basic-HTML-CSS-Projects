Linux Intro

- An operating system acts as a bridge between software and hardware.
- Without an operating system, software can't directly communicate with hardware.
- The user installs software on their server, which communicates with the operating system, which then communicates with the hardware.
- Examples of popular operating systems are Windows, Mac, and Linux.

Why Linux is Popular

- Linux is a free and open-source operating system.
- Linux is very secure and does not require antivirus software.
- Linux is fast and reliable, which is important for production systems.
- Popular Linux distributions include Ubuntu, CentOS, and Debian.

Operating System Fundamentals

- Operating system is responsible for establishing communication between hardware and software
- Linux is a popular operating system due to its free, secure, and fast nature
- Linux operating system architecture includes kernel, system libraries, compilers, user processes, and system software
- Kernel has primary responsibilities including device management, memory management, process management, and handling system calls

Shell Scripting Fundamentals

- Shell is a way to talk to the operating system
- Shell commands are used to interact with the Linux operating system
- Most servers default to a shell environment
- Shell commands are common across different Linux distributions
- Shell commands are used to create files, navigate through folders, and perform other tasks on the operating system.

Directories And Files

- To see the files and folders in a directory, use the command `ls`.
- To see the present working directory, use the command `pwd`.
- To change the directory, use the command `cd`.
- To go up one directory, use the command `cd ..`.
- To list all files and folders with their properties, use the command `ls -ltr`.

Create Directories And Files

- To create a file, use the command `touch` followed by the file name.
- To write into a file, use the command `echo` followed by the text to be written and redirect the output to the file using the `>` operator.
- To print the contents of a file, use the command `cat` followed by the file name.
- To create a directory, use the command `mkdir` followed by the directory name.
- To remove a file, use the command `rm` followed by the file name.
- To remove a directory, use the command `rm -r` followed by the directory name.

Memory,CPU and Disk Managment

- To check memory usage, use the command `free`.
- To check CPU usage, use the command `nproc`.
- To check disk usage, use the command `df`.
- To view all of these details in one place, use the command `top`.
